# Telegram Finance Bot

Простой телеграм-бот для учета доходов и расходов.

## Что умеет

- принимать быстрые записи в формате `-350 кофе` или `+50000 зарплата`
- хранить данные локально в файле `data/finance_data.json`
- показывать сводку за день и месяц
- считать общий баланс
- строить отчет по категориям расходов
- показывать последние 10 операций
- удалять последнюю операцию командой `/undo`

## Стек

- Python 3.13.12
- python-telegram-bot 22.7
- python-dotenv 1.1.1

## Запуск

### 1. Создай виртуальное окружение

```bash
python3.13 -m venv .venv
source .venv/bin/activate
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Создай `.env`

Скопируй `.env.example` в `.env` и вставь токен:

```bash
cp .env.example .env
```

Внутри `.env`:

```env
TG_BOT_API_KEY=твой_токен_бота
```

### 4. Запусти бота

```bash
python main.py
```

## Примеры сообщений

- `-350 кофе`
- `-1290 продукты - пятёрочка`
- `+120000 зарплата`
- `+5000 фриланс | лендинг`

## Команды

- `/start`
- `/help`
- `/today`
- `/month`
- `/balance`
- `/categories`
- `/last`
- `/undo`

## Где лежат данные

Все операции пишутся локально в:

```text
data/finance_data.json
```
